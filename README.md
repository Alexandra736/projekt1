# Projekt 1 część " Gra o snake"

Opis projektu: przedstawiam grę typu " snake", napisaną w języku C++ z użyciem SFML. Gra zawiera animowane objekty, obsługe klawiatury, zasis i odczyt stanu gry.

# Funkcjonalności
1. Sterowanie wężem za pomocą klawiatury,
2. Animowany ruch węża,
3. Losowo generowane jedzenie,
4. System punktów i rekordów,
5. Zapis i odczyt stanu gry do pliku,
6. Bonusy o nieregularnym kształcie,
7. System kolizji.

# Struktura projektu :
1. main.cpp – punkt wejścia programu
2. menu – menu główne gry
3. game – główna logika gry
4. waz – klasa węża
5. jedzenie – klasa jedzenia
6. bonus – system bonusów
7. efektytekstowe – animowane efekty tekstowe
8. menedzersave – zapis i odczyt stanu gry
